How it works

LinkedIn Integration: Your portfolio will automatically fetch your professional information from your LinkedIn profile, including:

  • Professional experience and job history
  • Education and certifications
  • Skills and endorsements
  • Profile summary and headline

GitHub Integration: Your projects section will display your public GitHub repositories, showing:

  • Repository names and descriptions
  • Programming languages used
  • Star and fork counts
  • Last updated dates

Note: This feature uses public APIs and may have rate limits. For production use, consider implementing proper API authentication.
